UI Script 과제: 전체 화면 메뉴와 모달 띄우기 과제 내용: 조건1: ‘메뉴 열기’ 버튼을 클릭하면 전체 화면 메뉴가 나타나야 합니다. 메뉴는 화면을 전체적으로 덮는 형태로 나타나야 하며, 애니메이션 효과를 포함해야 합니다. 조건2: 전체 화면 메뉴에는 ‘메뉴 닫기’ 버튼이 있어야 하며, 이를 클릭하면 메뉴가 애니메이션과 함께 사라져야 합니다. 조건3: ‘모달 열기’ 버튼을 클릭하면 모달 창이 화면 중앙에 애니메이션 효과와 함께 나타나야 합니다. 모달 창에는 ‘모달 닫기’ 버튼이 포함되어야 하며, 이 버튼을 클릭하면 모달이 사라집니다. 힌트: CSS **transition**과 transform 속성을 활용하여 부드러운 애니메이션을 구현할 수 있습니다. visibility 속성과 **opacity**를 사용하여 요소의 표시 및 숨김을 제어할 수 있습니다.
